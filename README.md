# Silon an E-commerce Template
A PSD to HTML project, It is a E-commerce psd template. It has multiiple section like header, offer, featured product, trending product and many more.
## Technology / Languages
1. HTML 5
2. CSS 3
3. Javascript
4. Boostrap
6. jQuery
